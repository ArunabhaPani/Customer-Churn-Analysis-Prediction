# 📊 Customer Churn Analysis & Prediction

An end-to-end **Customer Churn Analytics project** combining **SQL-based EDA, Machine Learning, and Power BI dashboards** to identify churn drivers and predict customer attrition.

---

## 🚀 Project Overview

Customer churn significantly affects business profitability.  
This project focuses on:
- Performing **EDA using SQL views in Microsoft SQL Server**
- Building **machine learning models** for churn prediction
- Creating **interactive Power BI dashboards** for business insights

---

## 🧠 Problem Statement

To analyze customer behavior and predict churn by leveraging:
- SQL-driven exploratory data analysis
- Machine learning classification models
- BI dashboards for decision-making

## 🧮 SQL-Based EDA (SSMS)

EDA was performed using **Microsoft SQL Server Management Studio** with:
- Creation of **SQL Views**
- Customer segmentation queries
- Churn rate analysis by tenure, contract, payment method
- Export of SQL views for Power BI integration

---

## 🤖 Machine Learning Workflow

1. Data extraction from SQL / Excel  
2. Data preprocessing & encoding  
3. Train-test split  
4. Model training  
5. Evaluation (Accuracy, Precision, Recall, F1-score)
